Parameters

limit

a limit on the number of objects to be returned. The Limit can range between 1 and 100, and the default is 20.

order

sort order by the createdAt timestamp of the objects. SortOrder.Ascending for ascending order and SortOrder.Descending for descending order.

after

a cursor for use in pagination. after is an object ID that defines your place in the list. For instance, if you make a list request and receive 100 objects, ending with obj_foo, your subsequent call can include after = AssistantId("obj_foo") in order to fetch the next page of the list.

before

a cursor for use in pagination. Before is an object ID that defines your place in the list. For instance, if you make a list request and receive 100 objects, ending with obj_foo, your subsequent call can include before = AssistantId("obj_foo") in order to fetch the previous page of the list.
