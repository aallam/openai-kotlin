# Sample App

JVM, JS and Native sample apps.

### Running the apps

| Target | command                                               |
| ------ | ----------------------------------------------------- |
| jvm    | `./gradlew :sample:jvm:run`                           |
| js     | `./gradlew :sample:js:run`                            |
| native | `./gradlew :sample:native:runReleaseExecutableNative` |

_PS: environment variable `OPENAI_API_KEY` must be set._
